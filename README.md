## 💡 术语科普与选购建议

为了帮助大家更好地选择适合自己的服务，这里简单解释一下不同线路的区别：

* **专线 (IEPL/IPLC)**：不过墙（GFW），不受封锁期影响，延迟极低，晚高峰表现非常稳定。适合对网络质量要求极高（如外服游戏、高端流媒体播放）的用户。价格相对较高。
* **中转 (BGP/公网隧道)**：通过国内优质服务器中转流量到海外，速度和稳定性大幅优于直连，性价比较高，是大多数日常办公和观影用户的**首选**。
* **直连**：直接连接海外服务器，受网络波动和墙的影响较大，晚高峰可能出现丢包卡顿。但价格极其便宜（如常说的“一元机场”），适合作为备用防失联节点或轻度网页浏览。
